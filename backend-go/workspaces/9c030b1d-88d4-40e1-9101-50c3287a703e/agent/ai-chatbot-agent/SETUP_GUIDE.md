# 🚀 Quick Setup Guide - Simple AI Chatbot

## Super Simple Setup (3 Steps!)

### ✅ Step 1: Check Python
Open your terminal/command prompt and type:
```bash
python --version
```
or
```bash
python3 --version
```

You should see Python 3.6 or higher. If not, download from: https://www.python.org/downloads/

### ✅ Step 2: Extract Files
Extract the zip file to any folder on your computer.

### ✅ Step 3: Run!
Open terminal in the extracted folder and run:

**On Windows:**
```bash
python main.py
```

**On Mac/Linux:**
```bash
python3 main.py
```

**That's it! No API keys, no installation, no setup!** 🎉

## ❓ Troubleshooting

### "python: command not found"
- Install Python from https://www.python.org/
- On Mac/Linux, try `python3` instead of `python`

### Script doesn't run
- Make sure you're in the correct folder
- On Mac/Linux, try: `chmod +x main.py` then `./main.py`

### Other issues
- Make sure `main.py` is in the same folder
- Try closing and reopening your terminal
